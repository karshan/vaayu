Required hardware:
   o MBHP_CORE_STM32 or STM32 Primer
   o an SD Card connected to J16 of the core module

===============================================================================

This is the Mackie C4 Pro MIDI translator project. It allows to use the Mackie
C4 Pro hardware controller to be used in stand-alone mode without the need for
the 'C4 Commander' software running on a PC or MAC. The application replaces
that completely.

MIDI In/Out 1 must be connected to the Mackie C4 Pro, MIDI In/Out 2 must be
connected to the device that is to be controlled.
